HTML Markup

Create an Unordered List with the <li>s as slides.
Optionally add a title attribute as a slide title / a lang attribute as an alternative slide interval.
See "setup" below.

CSS Settings

timeliner_styling.css contains all the styling with descriptive comments to help you along.

JavaScript Settings

When you call the plugin on your list element, you can supply options to overwrite the defaults. See an example under "Setup".
The JavaScript source is included and commented for more advanced manipulations.

Container:
containerwidth = Total width (value in pixels) (default = 800)
containerheight = Total height (value in pixels) (default 400)
showpauseplay = Whether to show the pause/play button when hovering the presentation (true or false) (default = true)
showprevnext = Whether to show the previous/next buttons when hovering the presentation (true or false) (default = true)
controls_always_visible = Always keep the pause/play/previous/next buttons visible (true or false) (default = false)
Timeline:
timelinewidth = Width of the timeline/progress bar (value in pixels) (default = 720)
timelineheight = Height of the timeline/progress bar (value in pixels) (default = 8)
timelinehorizontalmargin = Timeline left and right margin ('auto' (center) or number value in pixels) (default = 'auto')
timelineverticalmargin = Timeline top and bottom margin (number value in pixels) (default = 30)
timelineposition = Position the timeline at the top or bottom of the slide ('top' or 'bottom') (default = bottom)
Time display:
showtimedisplay = Whether to show the time display or not (true or false) (default = true)
timedisplayposition = Position the time display above or below the progress bar ('above' or 'below') (default = 'above')
showtotaltime = Whether to add the total presentation time after the current playing time (true or false) (default = false)
Node tooltip:
showtooltip = Whether to show the slide title in a tooltip when hovering a node (true or false) (default = true)
showtooltiptime = Whether to show the slide start time in a tooltip when hovering a node (true or false) (default = false)
tooltipposition = Position the tooltip above or below the progress bar ('above' or 'below') (default = 'above')
Playback:
interval = Time between slides in seconds, except when individually specified in the HTML (default = 10)
repeat = Whether to restart the presentation after completing (true or false) (default = true)
autoplay = Whether to start the presentation as soon as it loads (true or false) (default = true)
keyboard = Whether to use keyboard functionality (true or false) (default = true)
transition = Transition type between slides ('fade', 'instant', 'slide' or 'reveal') (default = 'slide')

Setup:

<!DOCTYPE HTML>
<html>
<head>
<!-- jQuery library: -->
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"></script>

<!-- Timeliner functionality CSS (Must be before the timeliner JavaScript): -->
<link rel="stylesheet" type="text/css" href="css/timeliner.css">
<!-- Timeliner styling CSS: -->
<link rel="stylesheet" type="text/css" href="css/timeliner_styling.css">
<!-- Timeliner JavaScript: -->
<script src="js/jquery.timeliner.min.js"></script>

<script>
$(document).ready(function(){
// Apply the script when the document is ready (minimum):
$('#element').timeliner();

// Or with custom options:
$('#element').timeliner({
containerwidth: 800,
containerheight: 400,
showpauseplay: true,
timelinewidth: 720,
timelineheight: 8,
timelinehorizontalmargin: 'auto',
timelineverticalmargin: 30,
timelineposition: 'bottom',
showtimedisplay: true,
timedisplayposition: 'above',
showtotaltime: false,
showtooltip: true,
showtooltiptime: false,
tooltipposition: 'above',
interval: 10,
repeat: true,
autoplay: true,
keyboard: true,
transition: 'slide'
});
});
</script>
</head>
<body>

<!-- HTML list element: -->
<ul id="your_element_id" class="timeliner">

// Minimal setup:
<li>Your slide content here</li>

// Full setup:
<li title="This slide's title" lang="(optional individual slide duration in seconds)">
Your slide content here
</li>

</ul>

</body>
</html>

Advanced: callback functions:

Function name Description
start_callback(id) Called when the presentation starts. This will pass the ID of the instance.
newslide_callback(id, slide) Called when a new slide is activated. This will pass the ID of the instance and the slide number.
end_callback(id) Called when the presentation ends. This will pass the ID of the instance.
paused_callback(id, slide) Called when the presentation is paused.
resumed_callback(id, slide) Called when the presentation is resumed.
click_callback(id, slide) Called when a slide is clicked (do not use if you are also using links in slides).

If you create the function outside the document-ready part of your javascript code, it will be called by the plugin when the appropriate event occurs.

Example:
$(document).ready(function(){
$('#element').timeliner();
});
// The presentation start callback:
function start_callback(id){
// Your custom events here
alert('Timeliner "'+id+'" started.');
}
